# Plugin Creator

Claude Code 插件创建向导，帮助生成 agents、commands、skills、hooks。

## 安装

```
/plugin add /path/to/plugin-creator
/reload-plugins
```

## 使用

```
/create-plugin
```

## 功能

- 创建新插件骨架
- 添加 Agent
- 添加 Command
- 添加 Skill
- 添加 Hook
- 验证插件配置

## 组件类型

| 类型 | 用途 |
|------|------|
| Agent | 自主执行复杂任务 |
| Command | 斜杠命令入口 |
| Skill | 知识库和最佳实践 |
| Hook | 事件钩子脚本 |
